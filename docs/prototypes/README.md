# Pare HTML prototypes

Click-through HTML mocks built during the discovery phase. They're the
canonical visual + interactive reference for the SwiftUI implementation.

| Version | Notes |
|---|---|
| v0.1 | Five core screens: Dashboard, Smart Scan, Duplicates, Cleanup Complete, Recovery Bin. |
| v0.2 | + onboarding flow, five category drill-downs, Settings (6 panels), Help, limited-mode banner, typed-confirmation modal. |
| v0.3 | + Profile, Subscription / Billing Center, Space Lens treemap, Startup Items, Downloads, Photos. |
| v0.4 | + auth flow: Splash, Sign Up, Sign In, Magic Link, Choose Plan. |
| v0.5 | + dark mode, NDPA/GDPR row, Dashboard storage sparkline, menu-bar status item, accessibility pass. |
| v0.6 | + Insights/Trends screen, sparkline range switcher (30d/90d/1yr) with tooltips, menu-bar state previews, keyboard shortcuts table. |
| v0.7 | Space Lens redesigned with a CleanMyMac-style purple bubble visualization (superseded by v0.8). |
| v0.8 | Space Lens rebuilt in Pare's own visual language and turned into a real folder browser: clickable breadcrumb, drill all the way down to individual files, protected paths visibly marked. Replaces v0.7. |

Open any of these in a modern browser at ≥ 1100 px wide. The pill bar at
the top of every prototype is a debug navigator — production navigation
lives in the sidebar and in-screen buttons. The theme toggle in v0.5+ at
the top-right cycles Light / Auto / Dark.
